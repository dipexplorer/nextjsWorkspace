# Next Multi Projects Workspace

[Short description]
A monorepo containing multiple Next.js projects and shared packages. Use this repository to develop, test, and deploy multiple Next.js applications from a single codebase.

## Contents

- `projects/` — individual Next.js applications (e.g. `projects/site-a`, `projects/site-b`)
- `packages/` — shared packages (UI components, utilities)
- `README.md` — this file

## Getting started

### Prerequisites

- Node.js (v16+ recommended)
- npm, yarn, or pnpm
- Git

### Install dependencies (root or per-project)

If you are using workspaces (optional):

```bash
# example with npm workspaces or yarn/pnpm
npm install
```
